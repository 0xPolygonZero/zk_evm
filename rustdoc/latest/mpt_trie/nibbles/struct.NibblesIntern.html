#[repr(C)]
pub struct NibblesIntern(pub [u64; 5]);
Expand description

Little-endian large integer type Used for the internal representation of a sequence of nibbles. The choice of [u64; 5] accommodates the 260-bit key requirement efficiently by leveraging 64-bit instructions for performance, while providing an additional u64 to handle the overflow case beyond the 256-bit capacity of [u64; 4].
