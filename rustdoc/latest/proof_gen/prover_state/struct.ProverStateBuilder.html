impl Default for ProverStateBuilder

source§

fn default() -> Self

Generates a new builder from a set of default ranges. These ranges should be sufficient to prove any transaction, but will require a significant amount of RAM (around 30GB).

Specifying shorter ranges will allow for a lower memory consumption, with the drawback of possibly not being sufficient for some transactions.
