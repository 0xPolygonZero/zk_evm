Statics§

  • CIRCUIT_VERSION
    When we serialize/deserialize circuits, we rely on the hash of the plonky kernel to determine if the circuit is compatible with our current binary. If the kernel hash of the circuit that we are loading in from disk differs, then using these circuits would cause failures during proof generation
